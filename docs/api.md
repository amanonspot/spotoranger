# API

FastAPI serves OpenAPI documentation at `/docs`.

Initial API groups:

- `GET /health`
- `POST /auth/otp/request`
- `POST /auth/otp/verify`
- `GET /properties`
- `GET /wallet/{ranger_id}`
- `GET /training`

- `/ranger`

- `/admin`
- `/notifications`
- `/withdrawals`
- `/maps`
